1). Create the Repo.
2). Upload the HTML and Logo file in Repo.
3). Wait for some time.
4). Goto Repo -> Setting -> Pages -> First one : Your site is live at "Link" or "Visit Site".
5). Copy link or Press the Visit Site.
6). Visit the QR Code generator site https://free-qr.com/
7). Paste the link and generate the QR Code and Download the QR Code.
